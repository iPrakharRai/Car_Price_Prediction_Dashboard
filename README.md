# 🚗 Car Price Prediction Dashboard

Welcome to the **Car Price Predictor**, a seamless web application built with Streamlit and Machine Learning to estimate the market value of used cars based on historical data.

## ✨ Features
* **Accurate ML Predictor**: Powered by a finely tuned Linear Regression model.
* **Modern UI**: Features a beautiful dark mode with glowing blue accents.
* **Clean Real-time Data**: Input your car's company, manufacturing year, kilometers driven, and fuel type to instantly get a dynamic price prediction!

## 🚀 Quickstart Guide

### 1. Clone the repository
```bash
git clone https://github.com/iPrakharRai/Car_Price_Prediction_Dashboard.git
cd Car_Price_Prediction_Dashboard
```

### 2. Install Dependencies
Make sure you have Python installed, then run the following to install the required libraries:
```bash
pip install -r requirements.txt
```

### 3. Run the Dashboard
Launch the Streamlit app locally:
```bash
streamlit run app.py
```
*The app will automatically pop up in your browser!*

## 📁 Repository Structure
- `app.py`: The main Streamlit web application script.
- `LinearRegressionModel.pkl`: The trained predictive model picking up on historical trends.
- `Cleaned_Car_data.csv`: The cleaned dataset used to populate options and train the model.
- `requirements.txt`: Python package dependencies required to run the project.
- `.streamlit/config.toml`: Custom theme configuration for the aesthetic dark/blue UI.
